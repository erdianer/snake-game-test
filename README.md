# 贪吃蛇小游戏

这是一个使用 HTML、CSS 和 JavaScript 编写的网页版贪吃蛇小游戏。  
玩家可以通过键盘方向键控制蛇的移动，吃到食物后得分增加，碰到墙壁或自身后游戏结束。

## 在线体验

点击下面链接可以直接游玩：

https://erdianer.github.io/snake-game-test/

## 项目功能

- 开始游戏界面
- 游戏运行界面
- 得分显示
- 游戏结束界面
- 重新开始功能
- 使用方向键控制蛇移动

## 操作方法

使用键盘方向键控制蛇的移动方向：

- ↑：向上移动
- ↓：向下移动
- ←：向左移动
- →：向右移动
